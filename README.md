# HTML Compiler

## Overview

HTML Compiler is a software tool designed to convert HTML files into standalone executable applications for Windows. It allows developers and users to package web content into a single executable file, facilitating easier distribution and use without requiring a web browser.

## Key Features

**Convert HTML files into standalone Windows executables**  
**Support for JavaScript, CSS, and multimedia content**  
**Customizable application window and behavior**  
**Integrated resource embedding for offline use**  
**Option to disable right-click, text selection, and keyboard shortcuts**  
**Simple interface for compiling and managing projects**  
**Support for multiple HTML pages within one executable**  

## Why HTML Compiler?

HTML Compiler offers a straightforward approach for transforming web projects into executable files, focusing on clarity and consistent performance. It provides a reliable method to package HTML content with embedded resources in a format that runs natively on Windows. The tool emphasizes usability by offering clear options and settings without unnecessary complexity.

## Benefits

HTML Compiler enables easy distribution of web-based applications without dependency on external browsers or internet connections. It helps ensure consistent presentation and functionality by embedding all necessary resources into a single executable. This approach simplifies deployment and reduces compatibility concerns across different Windows environments.

## Compatibility

This repository is developed specifically for Windows operating systems. It ensures stable performance and efficient operation within the Windows environment, taking advantage of native system features for application execution and resource management.
